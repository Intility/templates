## Prerequisites

- Some experience with React
- [node.js](https://nodejs.org/)
- [Visual Studio Code](https://code.visualstudio.com/) with the following plugins
  - [Node Extension Pack](https://marketplace.visualstudio.com/items?itemName=swellaby.node-pack)
  - [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

## Create App

Run these commands in your terminal.

```bash
npx degit intility/templates/react my-app
cd my-app

npm install
```

Open the project folder in Visual Studio Code. On windows, you can do so with the following command.

```bash
code .
```

You can now start coding by running the start script.

```bash
npm start
```

:::info
The template includes a `.npmrc` config which sets up a custom registry for `@intility/*` and `@fortawesome/*` packages.

For alternative package registry setup, see the [NPM docs in Backstage](https://backstage.apps.aa.intility.com/docs/default/component/package-registry/npm/).
:::
