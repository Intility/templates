The template includes some dependencies which are found to be common needs of most React apps by Intility.

## @intility/bifrost-react

- [Docs](https://bifrost.intility.no/)

Intility's design system for React.

## @azure/msal-\*

- [`@azure/msal-react` Docs](https://github.com/AzureAD/microsoft-authentication-library-for-js/blob/dev/lib/msal-react/docs/getting-started.md)
- [`@azure/msal-browser` Docs](https://github.com/AzureAD/microsoft-authentication-library-for-js/blob/dev/lib/msal-browser/docs/initialization.md)

Authentication libraries.

## react-router

- [Docs](https://reactrouter.com/en/main)

Unless you're a dashboard, you need a router, and react-router is the de facto router for React.

## @sentry/react and @sentry/tracing

- [Docs](https://docs.sentry.io/platforms/javascript/guides/react/)

Used for error reporting to Sentry.

## prettier

- [Docs](https://prettier.io/)

Prettier is a code formatter and great tool for unifying formatting when collaborating on a project.
The Visual Studio Code Workspace is set up to auto-format on save.
